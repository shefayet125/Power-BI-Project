# Super Store Sales Dashboard & Forecast (Power BI)
## Project Overview
This project showcases an interactive **Power BI dashboard** built using Super Store data. It provides deep insights into sales, profit, customer behavior, and shipping patterns, along with a **15-day sales forecasting model** for future planning.

### Key Performance Indicators (KPIs)

| Metric | Value |
|------|------|
| 💰 Total Sales | 1.57M |
| 📦 Total Quantity | 22K |
| 📈 Total Profit | 175K |
| 🚚 Avg Delivery Time | 4 Days |
